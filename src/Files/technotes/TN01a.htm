Tech Note 01a:
Manually Uninstalling NS Basic/CE

February 6, 2006

© 2010 NS BASIC Corporation. All rights reserved.
The Uninstaller for NS Basic/CE works in almost all cases. However, in the event of corruption of Windows or manual deletion of some files, it can fail. Here are instructions for completely uninstalling NS Basic/CE without a script. This should work with NS Basic/CE 5.0 and later. With earlier versions, the only difference will be in the Registry step.

1. Uninstall the program files

Delete the directory \program files\nsbasic\ce

2. Uninstall the helper files

In \Windows\System32 (or \WinNT\System32 in Windows 2000), delete the following files if they exist:

  • NSBAddScript.dll (only if you are not using NS Basic/Deskop)
  • NSBPhone.ocx
  • NSBPictB.ocx
  • NSCEPictureBox.ocx

3. Clear out Registry entries

Do Start->Run to bring up the Run menu.
Enter Regedit into the Open: field.
Navigate to HKEY_CURRENT_USER\Software\NSBasic
Delete the CE key (which will delete everything beneath it).

4. Clean up the Start menu

Right click on Start and Choose Open
Choose Programs
Right click on NS Basic CE and delete it.
