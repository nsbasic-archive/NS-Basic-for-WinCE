TIME                                                                       Function

TIME

Description

TIME returns the current system time.

Example

REM TIME Example

'TIME returns current system time

DIM RightNow

RightNow = TIME

PRINT "The time now is " & RightNow

Output

The time now is 10:52:44 PM

(sample time output is system dependant)
