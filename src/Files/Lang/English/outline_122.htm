SPACE                                                                    Function

SPACE(number)

Description

SPACE returns a string consisting of a number of spaces. The required argument, number, is any valid numeric expression.

Example

REM SPACE Example

'SPACE creates a string of spaces

DIM Spaces, Letter

Spaces = SPACE(4)

FOR i = 0 to 4

  Letter = LEFT(Spaces, i) & CHR(65 + i)

  PRINT Letter

NEXT

Output

A

 B

  C

   D

    E
